# Packages

This directory contains all packages for the Fintracker & Vault monorepo.

## Structure

- **shared/** - Shared packages used by all apps
- **apps/** - Application packages (fintracker, vault)
- **tools/** - Development tools and configurations

## Shared Packages

- `@fintracker-vault/ui` - Shared UI components and theme
- `@fintracker-vault/types` - Shared TypeScript types
- `@fintracker-vault/config` - Shared configuration
- `@fintracker-vault/utils` - Shared utilities

## Apps

- `fintracker` - Financial tracking application
- `vault` - Data vault application

## Tools

- `scripts/` - Build and utility scripts
- `configs/` - Shared configuration packages (eslint, prettier, tsconfig)
