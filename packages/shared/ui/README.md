# @fintracker-vault/ui

Shared UI components and theme system for Fintracker and Vault applications.

## Features

- Reusable React components
- Centralized theme configuration
- Tailwind CSS integration
- TypeScript support

## Usage

```typescript
import { Button, Card } from '@fintracker-vault/ui';
import { theme } from '@fintracker-vault/ui/theme';
import { useTheme } from '@fintracker-vault/ui/hooks';
```

## Structure

- `src/components/` - Reusable UI components
- `src/theme/` - Theme configuration
- `src/hooks/` - Theme-related hooks
- `src/styles/` - Global styles

## Building

```bash
pnpm build
```
