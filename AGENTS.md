# Repository Entry Point

This repository uses a single shared guide for all contributors and tools.

## Start Here

- Read `README.md` first.
- Use `CLAUDE.md` for concise repo guardrails and doc links.

## Required Workflow

1. Create a branch from `main`.
2. Implement changes.
3. Run `pnpm type-check`.
4. Push branch and open PR into `main`.

Do not push feature commits directly to `main`.
